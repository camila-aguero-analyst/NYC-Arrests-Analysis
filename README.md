# SQL Workflow

This folder contains the SQL workflow used to prepare and explore the NYPD Arrests dataset for Tableau visualization.

## 01_clean_arrests_data.sql

Creates a cleaned analysis table by:
- standardizing borough names
- handling missing values
- validating geographic coordinates
- extracting date fields

## 02_feature_engineering.sql

Creates derived variables used throughout the dashboard.

## 03_exploratory_analysis.sql

Contains exploratory queries used to identify trends by borough, demographic group, and time period prior to dashboard development.
